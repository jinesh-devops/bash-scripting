# Intermediate Level Bash Scripting

This folder contains practical Bash automation scripts commonly used in daily DevOps / Linux Admin / SRE operational work.  
These scripts move beyond fundamentals and start focusing on real environment observability, monitoring, alerting and system troubleshooting.

| Script Name                 | Purpose / Description                                                                 |
|-----------------------------|-----------------------------------------------------------------------------------------|
| check_ssl.sh                | Validates SSL certificate expiry date and warns before expiry                          |
| directory_backup.sh         | Creates directory backup and stores compressed archive                                  |
| disk_alert.sh               | Monitors disk usage and triggers alert when threshold is reached                        |
| find_top_large_files.sh     | Identify top largest files/folders inside a directory                                   |
| inactive_users.sh           | Detect users inactive for long time in Linux system                                     |
| log_error_analyzer.sh       | Scan log files and extract errors, warnings, critical events                             |
| service_monitor.sh          | Monitors a given service and alerts if service is down                                  |
| system_report.sh            | Generate complete system health report (CPU, Memory, Disk, Services etc)                |
| top_mem_process.sh          | Lists top memory consuming processes                                                    |
| url_health_check.sh         | Checks availability for given URL(s) and generates pass/fail health status              |

### Capabilities Developed at Intermediate Level

- Linux OS operations automation
- Monitoring + alert style bash scripting
- System troubleshooting automation logic
- Error/Log pattern parsing
- Backup handling automation
- Threshold based alert scripting techniques
